# Frontend

[React](https://reactjs.org/) documentation.
[Vite](https://vitejs.dev/) documentation.

## Compile and run the project

```bash
# development
$ pnpm run dev

# production mode
$ pnpm run build
```

## Run tests

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
